# poc_cypress_with_priotag_n_report
A basic POC showing batch script for unix and windows, prio tag and html report 

## Start server with 

**npm run start**


## Start Build Verify Tests, BVT tests 

### Windows
**./runtests.cmd bvt**

### Linux / MacOS
**./runtests.sh bvt**

(execute chmod +x runtests.sh first)
